# 52x52 Dota 2 Statisctics

This repository is intended solely for entertainment and statistical purposes.

At the moment the project only contains the following data for display:

- **ALL** - statistics of all players for all games;
- **CARRY** - - statistics of all players in the Carry position for all games;
- **MIDER** - - statistics of all players in the Mider position for all games;
- **HARDER** - - statistics of all players in the Harder position for all games;
- **SUPPORT** - - statistics of all players in the Support position for all games;
- **HARD_SUPPORT** - - statistics of all players in the Hard Support position for all games;
